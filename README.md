#Quiz System Project

This project is a Java-based application designed to manage quiz questions through file I/O operations and automated scoring.

##Key Features

* **File Operations:** The system reads questions from "questions.txt" and exports final results to "results.txt".
* **Question Types:** Supports Multiple Choice and True/False questions using inheritance.
* **Score Management:** Student information and quiz performance are handled and stored automatically.

##Testing & Documentation
* **JUnit5:** Core logic and data encapsulation are verified with unit tests.
* **Javadoc:** Technical documentation is generated and available in the 'doc/' folder.
